# Vault 目录结构建议

> 这是推荐的 Obsidian Vault 目录结构，你可以根据自己的需求调整。

```text
你的Vault/
├─ 01_内容创作/
│  ├─ 00_内容创作规范/     ← 系统规则文件放这里
│  ├─ 01_选题池/           ← 灵感和选题记录
│  ├─ 02_内容生产中/       ← 正在写的文章
│  └─ 03_已发布内容/       ← 写完的按月归档
├─ 02_资料库/              ← 长期知识资产
├─ 03_日记复盘/            ← 个人复盘
└─ 04_学习输入/            ← 外部学习资料
```

## 每篇文章的目录结构

```text
01_内容创作/02_内容生产中/YYYYMMDD-文章标题/
├─ 00_内容卡片.md     ← 文章元数据（标题、摘要、状态等）
├─ 01_文章内容/
│  ├─ 公众号文章.md    ← 文章正文（需有 frontmatter: title + description）
│  ├─ 配图建议.md      ← 配图提示词和截图指引
│  └─ images/          ← 文章配图
├─ 02_视频素材/        ← 转视频时用
├─ 03_封面图/          ← 封面图
├─ 04_最终视频/        ← 视频成片
└─ 05_发布文案/        ← 跨平台发布文案
    └─ 跨平台发布文案.md
```
