# 风格：简约手绘

白底黑线手绘质感，轻松易懂，适合概念解释和流程说明。

## 适用场景
- 文章类型：概念解释、入门教程、流程指南、分类说明
- 氛围：轻松、亲切、易理解、无距离感

## 视觉特征
- 画面质感：手绘线条质感，轻微不规则，有人工痕迹
- 色彩特点：白底为主，黑色线条，可用一种点缀色
- 构图特点：节点式布局，简洁箭头连接，留白充足
- 文字风格：无衬线中文字体，少量关键词

## 全局风格指令

生成提示词时，每个提示词前缀必须包含以下描述：

```
简约手绘风格插画，纯白背景，黑色线条带有轻微手绘质感。简洁几何图形通过清晰的线条和箭头连接，形成逻辑图表。极少文字，仅保留1-3个核心中文关键词。无真实照片元素，无渐变色。
```

## 配色方案
- 主色：黑色（线条、文字）
- 辅色：深灰（轮廓、次要元素）
- 背景：纯白
- 点缀色：可选一种低饱和度颜色（如淡蓝、淡绿）

## 风格总结

- 简约手绘风格，适合概念解释、流程说明、入门教程
- 白底黑线，可有一种点缀色
- 手绘线条质感，轻微不规则
- 节点式布局，箭头连接
- 极少文字，1-3个中文关键词
- 亲切轻松，降低阅读门槛
