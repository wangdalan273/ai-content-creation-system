# 风格：建筑蓝图

精致极简、技术期刊感、高端专业，适合系统架构和深度技术分析。

## 适用场景
- 文章类型：系统架构、技术方案选型、深度技术分析、数据流分析
- 氛围：专业、精准、高端、理性

## 视觉特征
- 画面质感：精确几何线条，细微网格底纹，高端技术期刊排版感
- 色彩特点：白底浅灰网格，深炭灰线条，一种点缀色标记重点
- 构图特点：等距排列模块，层次分明，精确对齐
- 文字风格：精致无衬线字体，专业排版

## 全局风格指令

生成提示词时，每个提示词前缀必须包含以下描述：

```
建筑蓝图编辑风格图表，纯白背景带浅灰网格线，深炭灰色精确线条。精致的极简技术期刊排版风格，几何精度，专业感。极少文字，仅保留1-3个核心中文关键词。无真实照片元素，无渐变色。
```

## 配色方案
- 主色：深炭灰 #333333（线条、文字）
- 辅色：中灰（网格线、次要元素）
- 背景：纯白 + 浅灰网格
- 点缀色：一种精准色彩（如深蓝、深红）
