# 风格：单色流程

单色流程风格，黑白灰配色，强调流程化、系统化、逻辑性，适用于技术教程、流程指南、系统架构。

## 适用场景
- 文章类型：技术教程、流程指南、系统架构、技能说明、工具介绍
- 氛围：冷静、专业、理性、高效

## 视觉特征
- 画面质感：线条简洁、无多余装饰、手绘涂鸦感
- 色彩特点：黑白灰单色系、无彩色干扰
- 构图特点：流程化布局（线性/环形/模块化）、箭头引导、层级清晰
- 文字风格：无衬线体、等宽字体、简洁直接

## 全局风格指令

生成提示词时，每个提示词前缀必须包含以下描述：

```
单色流程风格图表，纯白背景，黑白灰配色，简洁线条。手绘涂鸦质感，箭头引导流程方向，层级清晰。极少文字，仅保留1-3个核心中文关键词。无真实照片元素，无渐变色。
```

## 配色方案
- 主色：黑色（线条、文字）
- 辅色：深灰（轮廓、阴影）
- 背景：白色或浅灰，白色为主
- 点缀色：无
