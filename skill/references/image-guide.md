# 配图指南

## 核心思路

不是"每隔N字配一张图"，而是**分析文章结构，为每个关键段落匹配合适的图表类型，生成可直接用于即梦的中文提示词**。

---

## 工作流程

### Step 1：文章结构分析

读取已完成的文章，按以下规则拆分：

- 以 `## ` (H2) 或 `### ` (H3) 标题作为段落分隔符
- 忽略纯引用、脚注等辅助内容
- 提取每个段落的：标题、核心观点（1句话）、关键概念（3-5个词）

输出格式：

```
## 文章结构分析

| # | 段落标题 | 核心观点 | 关键概念 | 推荐图表类型 |
|---|---------|---------|---------|------------|
| 1 | xxx     | xxx     | A, B, C | 概念图 |
| 2 | xxx     | xxx     | D, E, F | 流程图 |
| 3 | xxx     | xxx     | G, H, I | 对比图 |
```

### Step 2：图表类型匹配

根据段落内容自动匹配最合适的图表类型：

| 图表类型 | 适用内容 | 识别信号 |
|---------|---------|---------|
| **概念图** | 解释一个概念及其组成部分 | "是什么"、"包含"、"由...组成" |
| **流程图** | 有先后顺序的步骤/过程 | "步骤"、"流程"、"首先...然后"、"→" |
| **对比图** | 两个或多个事物的差异 | "vs"、"对比"、"区别"、"A和B" |
| **层次图** | 有上下级/包含关系 | "架构"、"层次"、"底层"、"顶层" |
| **关系图** | 多个元素之间的关联 | "关系"、"影响"、"连接"、"相互作用" |

**按文章风格优先匹配：**

| 文章风格 | 优先图表类型 | 原因 |
|---------|-----------|------|
| B 技术博客 | 流程图、层次图 | 步骤展示、架构说明 |
| D 评测对比 | 对比图、概念图 | 多维度对比、场景分析 |
| E 观点洞察 | 关系图、概念图 | 概念关联、逻辑推演 |

### Step 3：生成全局风格指令

在为每个段落生成提示词之前，先确定统一的全局风格指令。

**读取 `assets/配图风格/` 目录下所有风格文件**，展示给用户选择：

```
检测到以下配图风格，请选择：
1. 单色流程 — 黑白灰，流程化，冷静专业
2. 简约手绘 — 白底黑线，手绘质感，轻松易懂
3. 建筑蓝图 — 精致极简，技术期刊感，高端专业

直接回复数字选择，或回复风格名称。
```

选定后，读取对应风格文件，该文件定义了全局视觉基调和提示词前缀。

### Step 4：生成提示词

为每个段落生成配图提示词，每个提示词包含：

1. **风格前缀** — 来自所选风格文件的全局描述
2. **图表类型指令** — 概念图/流程图/对比图/层次图/关系图
3. **内容描述** — 以段落核心思想为主，叙述性描述图表内容
4. **中文标签** — 直接写中文关键词（如"数据采集"、"智能处理"），1-3个
5. **排除项** — 无照片元素、无渐变、无真实场景

展示格式（等待用户确认）：

```
## 配图建议大纲

### 图 1: [段落标题]
- 图表类型: 概念图
- 对应段落: 第2节
- 提示词:
  """
  [完整中文提示词]
  """

### 图 2: [段落标题]
...

以上提示词是否满意？是否需要修改某个图的描述？
确认后将生成完整的配图建议文档。
```

### Step 5：生成配图建议文档

用户确认后，生成配图建议文档保存到文章目录下。

---

## 提示词构建规则

### 规则

1. **中文描述为主** — 提示词用中文写，即梦对中文理解更好
2. **叙述性描述** — 用完整句子描述画面，不是关键词堆砌
3. **明确指定** — 白色背景、横版、极少文字、无真实照片
4. **具体化视觉元素** — 箭头方向、节点形状、布局方式都要说清楚
5. **中文标签限制** — 1-3个核心关键词，直接写中文
6. **一致性** — 同一篇文章所有配图使用相同风格前缀
7. **无技术参数** — 不写像素、分辨率、文件格式等制作指令

### 提示词模板结构

```
[风格前缀]。[图表类型]。
[内容描述，包含节点、连线、标签的具体位置和关系]。
图中仅保留以下中文标签："关键词1"、"关键词2"、"关键词3"。
不要真实照片、不要渐变、不要复杂场景。
```

## 封面图特殊规则

每篇文章必须有封面图，**不按段落分析**，单独生成。

- **尺寸**：21:9 横版
- **内容**：⚠️ **文章标题必须作为封面图的视觉主体**，用较大的手写风格中文字体呈现，融入画面构图。标题文字是封面的焦点，辅以象征性画面元素。
- **提示词**：必须在提示词中明确写出完整的文章标题文字，并指定"标题文字作为画面视觉焦点/主体"
- **常见错误**：❌ 只画抽象图案不放标题文字 → ✅ 标题文字居中/居上 + 下方配简笔画面元素

---

## 尺寸规范

| 类型 | 比例 | 用途 |
|------|------|------|
| 封面图（必需） | 21:9 | 公众号首图 |
| 内容插图 | 16:9 | 文中配图 |
